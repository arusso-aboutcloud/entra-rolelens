How it works

  1. Type any Microsoft Entra admin task in plain language
  2. Results show the minimum built-in role required
  3. Click any result to see the full role permissions
  4. Use Role Diff to compare any two roles side by side

Results are sourced directly from Microsoft's official "least privileged by task" documentation, refreshed nightly. No AI — every result is deterministic.

Compare permissions between any two roles

Select any two Entra ID built-in roles to compare their permissions side by side. See exactly what one role can do that the other cannot.

How to use

How to use: Type a role name in each field — autocomplete will suggest matching roles. Or click one of the example comparisons above to get started instantly.

  • Left column (red) — permissions only Role A has
  • Center column (gray) — permissions both roles share
  • Right column (green) — permissions only Role B has

Tip: compare a broad role (Global Administrator) against a scoped role (User Administrator) to find the minimum role for your use case.
